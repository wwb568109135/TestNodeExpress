
# express 3.x  + node + mongoodb 


一个简单的电商网站demo，使用了nodejs, express, mongodb, mongoose等功能模块。

实现功能：

1. 注册 
* 增加用户记录到数据库
 
2. 登录
* 去数据库查记录，验证用户登录

3. 添加商品
* 增加商品记录到数据库

4. 加入购物车和结算
* 删除和更新记录购物车记录到数据库


使用说明：

1. 启动mongodb（自己部署，自己百度）；

2. package.json 有配置文件 ：在命令行窗口 cd 到项目目录，
* npm install

3. 启动项目: 输入: node app.js

4. 浏览器通过 http://localhost:3000 访问了。
